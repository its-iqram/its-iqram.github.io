### Problem 15: Lattice Paths
- **Question**: How many routes through a 20×20 grid?
- **Solution**: $\binom{40}{20} = \frac{40!}{20! \cdot 20!}$
- **Code**: Used dynamic programming approach

```python
def lattice_paths(n):
    dp = [[0] * (n + 1) for _ in range(n + 1)]
    for i in range(n + 1):
        dp[i][0] = dp[0][i] = 1
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            dp[i][j] = dp[i-1][j] + dp[i][j-1]
    return dp[n][n]
